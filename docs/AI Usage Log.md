# 🤖 AI Usage Log

## 📄 Project
Smart Waste Classification System using MobileNetV2

---

# 📌 Purpose of This Document
This document records how Artificial Intelligence (AI) tools were used during the development of this project. It explains where AI assistance was utilized, what was learned from the process, and how much of the final implementation was created independently versus AI-assisted.

---

# 🧠 AI Tools Used
- ChatGPT (OpenAI)
- Google Colab AI suggestions
- TensorFlow/Keras Documentation
- Kaggle Dataset Resources

---

# 🎯 Areas Where AI Was Used

## 1. Understanding Deep Learning Architectures
AI assistance was used to better understand:
- Convolutional Neural Networks (CNNs)
- Transfer Learning concepts
- MobileNetV2 architecture
- Image classification workflows
- Feature extraction using pretrained models

### Key Learning
We learned how pretrained models can significantly improve performance and reduce training time compared to building CNNs from scratch.

---

## 2. Code Generation Assistance
AI helped generate:
- Initial TensorFlow/Keras boilerplate code
- Data preprocessing pipelines
- Image augmentation setup
- Model training loops
- Evaluation and plotting code
- Prediction functions for uploaded images

### Key Learning
We learned how to structure a professional deep learning workflow, including training, validation, evaluation, and prediction phases.

---

## 3. Debugging and Error Resolution
AI tools assisted with debugging:
- Dataset path issues
- ZIP extraction problems in Google Colab
- TensorFlow import errors
- Shape mismatch issues
- Incorrect class labeling
- Training configuration problems

### Key Learning
We improved our debugging skills and gained experience troubleshooting common machine learning implementation issues.

---

## 4. Documentation and Project Organization
AI was used to help prepare:
- README.md
- requirements.txt
- GitHub repository structure
- Technical documentation
- Architecture explanations
- Project summaries

### Key Learning
We learned how to present AI projects professionally using industry-standard documentation practices.
